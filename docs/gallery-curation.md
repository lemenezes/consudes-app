### Schema Supabase sugerido

```sql
-- Álbuns da galeria
CREATE TABLE gallery_albums (
  id          uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  slug        text UNIQUE NOT NULL,           -- ex: 'interclubes/2018-ic'
  title       text NOT NULL,                  -- ex: 'IC 2018'
  description text,
  year        integer,                        -- ex: 2018
  category    text,                           -- 'interclubes' | 'juegos-sudamericanos' | ...
  cover_path  text,                           -- path relativo no R2
  photo_count integer DEFAULT 0,
  is_published boolean DEFAULT false,
  sort_order  integer DEFAULT 0,
  created_at  timestamptz DEFAULT now()
);

-- Fotos de cada álbum
CREATE TABLE gallery_photos (
  id           uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  album_id     uuid REFERENCES gallery_albums(id) ON DELETE CASCADE,
  filename     text NOT NULL,
  storage_path text NOT NULL,                 -- path completo no R2
  alt_text     text,
  is_cover     boolean DEFAULT false,
  sort_order   integer DEFAULT 0,
  width        integer,
  height       integer,
  file_size    integer,                       -- bytes
  created_at   timestamptz DEFAULT now()
);

-- Index para queries por álbum
CREATE INDEX idx_photos_album ON gallery_photos(album_id, sort_order);
CREATE INDEX idx_albums_category ON gallery_albums(category, year);
```

### Lazy loading e otimização

- **Thumbnails**: Gerar via Cloudflare Image Resizing (`?width=400&fit=cover`) ou via transformação no upload
- **Hero/capa**: Servir em 800–1200px via parâmetro de URL
- **Imagens originais**: Preservar no R2 sem compressão
- **IntersectionObserver**: Usar `loading="lazy"` no `<img>` ou componente `<Image>` do React

### Estratégia de upload para R2

1. Otimizar imagens localmente com `sharp` ou `squoosh` antes do upload
2. Especialmente: `Logo_Futsal_2017.png` (7,5 MB → converter para WebP, ~200 KB)
3. Upload via CLI `wrangler r2 object put` ou script Node.js com AWS SDK v3 (R2 é S3-compatible)
4. Após upload: popular tabela `gallery_photos` no Supabase com paths e metadados
